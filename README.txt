Требования к системе:
Python 3.x должен быть установлен на вашем компьютере.
Установленные модули: Flask, Flask-SQLAlchemy, Flask-Admin, requests, packaging.
Доступ к Интернету для выполнения HTTP-запросов между клиентом и сервером.
Инструкция по запуску и использованию системы:
1. Запуск сервера:
Откройте командную строку или терминал.

Перейдите в каталог, содержащий серверный код с помощью команды:

cd путь_к_ващему_файлу


Запустите сервер, выполнив следующую команду:

python Server.py


2. Запуск клиента:
Откройте новое окно командной строки или терминала.

Перейдите в каталог, содержащий клиентский код с помощью команды:

cd путь_к_ващему_файлу

Запустите клиент, выполнив следующую команду:

python Client.py

После запуска клиента, он выполнит HTTP-запросы к серверу для получения правил безопасности и их проверки на устройстве. Результаты проверки будут выведены в консоли.

Доступ к панели администратора:
После запуска сервера, вы можете получить доступ к панели администратора, перейдя по следующей ссылке в веб-браузере:

http://127.0.0.1:5000/admin/

Панель администратора предоставляет интерфейс для просмотра и управления данными моделей SecurityRule(то где создаются правила), SecurityLog(то где показывается id устройство и выполнило ли оно правило или нет) и RuleTriggeredDevices.
